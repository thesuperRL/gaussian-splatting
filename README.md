# Gaussian Splatting Replication

A from-scratch reimplementation of Gaussian Splatting ([3DGS](https://arxiv.org/abs/2308.04079), [2DGS](https://arxiv.org/abs/2403.17888)) as **2D image splatting** in pure PyTorch

Implemented manually, with tab autocomplete turned off, and Cursor only queried for hints.
